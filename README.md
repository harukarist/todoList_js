# todoList_js

jQuery, Backbone.js, React.js/Redux, Vue.js を使った
4種類のToDoリストです。

1日の終わりに翌日のToDoを書き出す際に睡眠に悪影響を与えないよう、ダークモードを意識した配色としています。
